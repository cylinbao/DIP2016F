# White Balance
Related matalb script file: "wb.m"
In the matlab shell, execute "wb".
It will read input1.bmp and apply white balance, then generate the output named "input1_wb.bmp"

# Color Transformation
Related matalb script file: "ct.m"
In the matlab shell, execute "ct".
It will read "input2.bmp" and transfer the scene to autumn-like scenery, then generate the output named "input2_ct.bmp".
